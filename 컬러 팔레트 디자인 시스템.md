## CSS 변수 정의

```css
:root {
  /* Primary Colors */
  --color-primary:       #4AAFF2;  /* 메인 하늘색 */
  --color-primary-light: #8CCEF5;  /* 밝은 하늘색 */
  --color-primary-dark:  #2D8FD4;  /* 깊은 하늘색 */

  /* Secondary Colors */
  --color-secondary: #3A9EE0;  /* 보조 파랑 */
  --color-accent:    #B8DCEF;  /* 연한 구름 블루 */

  /* Background Colors */
  --color-bg-primary:   #FFFFFF;  /* 흰색 배경 (메인) */
  --color-bg-secondary: #E8F4FC;  /* 연한 하늘빛 배경 */

  /* Text Colors */
  --color-text-primary:   #1A3A52;  /* 진한 네이비 (본문) */
  --color-text-secondary: #2B6A99;  /* 중간 블루 (부제목) */
  --color-text-muted:     #6FA8C8;  /* 흐린 블루 (보조 텍스트) */

  /* Interactive Colors */
  --color-button-primary: #4AAFF2;  /* 기본 버튼 */
  --color-button-hover:   #2D8FD4;  /* 버튼 hover */
  --color-link:           #3A9EE0;  /* 링크 기본 */
  --color-link-hover:     #1A6FAA;  /* 링크 hover */
}
```

## 컬러 사용 가이드

### Primary (`#4AAFF2`)
- 히어로 섹션 배경, 헤더 배경
- CTA 버튼, 주요 아이콘
- 브랜드를 대표하는 핵심 요소에만 사용

### Primary Light (`#8CCEF5`)
- 카드 hover 효과, 선택된 항목 강조
- 배경 그라디언트의 밝은 끝
- 태그, 뱃지 배경

### Primary Dark (`#2D8FD4`)
- 버튼 hover / active 상태
- 배경 그라디언트의 어두운 끝
- 헤더 하단 경계선

### Secondary (`#3A9EE0`)
- 서브 섹션 제목 색상
- 아이콘 강조
- 구분선 색상

### Accent (`#B8DCEF`)
- 카드 테두리
- 입력 필드 focus 테두리
- 구분용 배경 (연한 영역)

### Background Primary (`#FFFFFF`)
- 전체 페이지 기본 배경
- 카드, 모달 배경

### Background Secondary (`#E8F4FC`)
- 섹션 구분용 연한 배경
- 사이드바, 푸터 배경

### Text Primary (`#1A3A52`)
- 본문 텍스트, 제목
- 고대비가 필요한 모든 텍스트

### Text Secondary (`#2B6A99`)
- 서브 제목, 캡션
- 부가 설명 텍스트

### Text Muted (`#6FA8C8`)
- placeholder, 비활성 상태 텍스트
- 날짜, 메타 정보

## 추천 그라디언트

```css
/* 하늘 그라디언트 (히어로 섹션용) */
background: linear-gradient(180deg, #2D8FD4 0%, #4AAFF2 50%, #8CCEF5 100%);

/* 부드러운 배경 그라디언트 */
background: linear-gradient(135deg, #E8F4FC 0%, #FFFFFF 100%);

/* 버튼 그라디언트 */
background: linear-gradient(90deg, #3A9EE0 0%, #4AAFF2 100%);
```

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary:     #0D1F2D;  /* 딥 다크 네이비 */
    --color-bg-secondary:   #162637;  /* 약간 밝은 다크 */
    --color-text-primary:   #E8F4FC;  /* 밝은 텍스트 */
    --color-text-secondary: #8CCEF5;  /* 연한 하늘 텍스트 */
    --color-text-muted:     #5A8FAA;  /* 흐린 텍스트 */
    --color-accent:         #1E4A6A;  /* 다크 테두리 */
    /* Primary 계열은 유지 (브랜드 컬러 일관성) */
  }
}
```

### 고대비 모드

```css
@media (prefers-contrast: high) {
  :root {
    --color-text-primary:   #000000;
    --color-button-primary: #0066CC;
    --color-link:           #0044AA;
  }
}
```

### 화면 크기별 색상 전략
- **모바일**: 배경을 `#FFFFFF`로 단순화, Primary 컬러를 포인트로만 활용
- **태블릿**: 사이드바에 `--color-bg-secondary` 적용
- **데스크탑**: 히어로 영역에 하늘 그라디언트 풀스크린 적용
